Abstract High Order Function in CSharp

具有相似功能的函数通常具有相似的结构，将相似的部分抽象出高阶函数，可以简化代码，提高复用率。

Original

如下面三个将字符串转化到其它类型的函数，其代码结构相似。

private int ConvertStr2Int(string s)
{
    if (string.IsNullOrEmpty(s)) return 0;
    int res = 0;
    if (int.TryParse(s, out res)) return res;
    return 0;
}

private DateTime ConvertStr2Dt(string s)
{
    if (string.IsNullOrEmpty(s)) return DateTime.Now;
    DateTime dt;
    try
    {
        dt = Convert.ToDateTime(s);
    }
    catch (Exception e)
    {
        return DateTime.Now;
    }
    return dt;
}

private bool ConvertStr2Bool (string s)
{
    if (string.IsNullOrEmpty(s)) return false;
    if (s.Equals("1")) return true;
    return false;
}
Problems
  • different empty value or exception value (add parameters?)
  • need convert string to other types (write one more similar function?)
  • different convert logic (how to name a similar function?)
  • source type is not string (how to cover this?)

One step ahead

将其中相似的部分抽象出函数 ConvertStr，string 到不同数据类型的转换逻辑作为函数的参数。

private T ConvertStr<T>(string s, T emptyValue, T exceptValue, Func<string, T> converter)
{
    if (string.IsNullOrEmpty(s)) return emptyValue;
    T res = exceptValue;
    try
    {
        res = converter(s);
    }
    catch
    {
        return exceptValue;
    }
    return res;
}

这样，以上三个字符串转换函数可以写为如下形式

private int ConvertStr2Int(string s)
{
    return ConvertStr<int>(s, 0, 0, (ss) => int.Parse(ss));
}

private DateTime ConvertStr2Dt(string s)
{
    return ConvertStr<DateTime>(s, DateTime.Now, DateTime.Now, (ss) => Convert.ToDateTime(ss));
}

private bool ConvertStr2Bool (string s)
{
    return ConvertStr<bool>(s, false, false, (ss) => ss.Equals("1"));
}

可以看出，原有的三个转换函数只是高阶函数部分参数化后的结果。

Furthermore

以上高阶函数只能实现从 string 到其它类型的转换。 进一步，可以将 string 类型及其控制的判定抽象出来，创建从 T1 转换到 T2 的高阶函数 MyConverter 。同样，函数 ConvertStr 是 MyConverter 部分参数化的结果

private T2 MyConverter<T1, T2>(T1 s, Func<T1, bool>isEmpty,  T2 emptyValue, T2 exceptValue, Func<T1, T2> converter)
{
    if (isEmpty(s)) return emptyValue;
    T2 res;
    try
    {
        res = convert(s);
    }
    catch
    {
        return exceptValue;
    }
    return res;
}

private T ConvertStr<T>(string s, T emptyValue, T exceptValue,Func<string, T> converter)
{
    return MyConverter<string, T>(s, (ss) => string.IsNullOrEmpty(ss), emptyValue, exceptValue, converter);
}
